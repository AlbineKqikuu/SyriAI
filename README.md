# SyriAI - AI Teleprompter

A professional, AI-powered teleprompter designed for news broadcasting and Master's thesis presentations. Features include real-time gaze correction (via MediaPipe), breaking news tickers, a digital studio clock, and a mirror mode for physical teleprompter hardware.

## Features
- **AI Gaze Tracking:** Uses MediaPipe Face Mesh to ensure you're always "looking" at the camera even while reading.
- **Dynamic News Ticker:** Customizable breaking news feed at the bottom of the screen.
- **Studio Interface:** High-contrast, easy-to-read typography with adjustable font sizes.
- **Broadcast Tools:** Built-in digital clock and connection status badges for a professional look.

## How to Run Locally
1. Clone or download this folder.
2. Open `index.html` in any modern web browser.
3. Allow camera access for the AI features to work.

## Hosting Online
To host this project so it can be accessed anywhere:
1. **Netlify (Drag & Drop):** Go to [netlify.com](https://www.netlify.com), login, and drag this entire folder into the upload area.
2. **GitHub Pages:** Upload these files to a new GitHub repository, then enable "Pages" in the repository settings.

## Technologies Used
- HTML5 / CSS3 (Modern Glassmorphism Design)
- JavaScript (Vanilla)
- [MediaPipe Face Mesh](https://google.github.io/mediapipe/solutions/face_mesh) for AI tracking.
